Affectris

A Tetris game with basic facial expression recognition — adapting the learning experience in real time based on detected affective states.

The Problem

My Role
Learning Experience Designer · Graduate Coursework, NYU
The Outcome
A playable game prototype that detects facial expressions via webcam and adjusts difficulty and feedback in real time based on detected emotional state.
