Cognizance

A Socratic Method web application for learning — exploring how dialogue-based pedagogy can be scaffolded through digital interaction design.

The Problem
How do you translate the Socratic Method — a profoundly relational form of inquiry — into a digital learning experience that still does what it's supposed to?
My Role
Learning Experience Designer · Graduate Coursework, NYU
The Outcome
A web application that scaffolds Socratic dialogue as an interactive learning experience, exploring how pedagogical questioning can be designed into digital interfaces.
